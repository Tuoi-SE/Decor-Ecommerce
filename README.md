## üöÄ Embark on the Journey

Transform your local environment into the Decor Shop development hub.

### 1. Acquire the Source
```bash
git clone https://github.com/yourusername/decor-shop-system.git
```

### 2. Ignite the Engines
We utilize Docker to orchestrate our infrastructure instantly.
```bash
docker-compose up -d
```

### 3. Launch the Satellite Services

**Server Command Center:**
```bash
cd server && npm install && npm run start:dev
```

**Client Interface:**
```bash
cd client && npm install && npm run dev
```
